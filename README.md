# 学习tmux

## 介绍

[tmux](http://tmux.github.io/)是一个终端复用软件.

> **什么是终端复用?**
>> tmux可以让你在一个终端里, 打开好几个程序, 在它们之间切换相当方便.
>> 你也可以随时把它们放到后台运行, 当有需要的时候, 再把它们呼唤出来.
>> 当然, 这只是tmux众多功能的冰山一角. 好戏还在后头...
